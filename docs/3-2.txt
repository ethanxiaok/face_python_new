构建后端项目目录

采用MTV思想

|---- manage.py # 入口启动文件
|---- app # 应用包
|-------- __init__.py # 初始化模块
|-------- configs.py # 配置信息模块
|-------- urls.py # 路由和视图映射的模块
|-------- models # 存放模型的包
|-------- templates # 存放模板的目录
|-------- admin # 存放后台管理系统的视图
|-------- api # 存放和前端交互API的视图
|-------- common # 公共包：人脸识别模块、IP地址解析、表单验证
|-------- static # 静态资源目录（css、js、图片、视频、媒体、动画）
|-------- ui # UI包，自定义一些UI组件


启动：python manage.py
